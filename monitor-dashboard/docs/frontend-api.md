# 前端接口文档（路由管理）

## 1. 获取路由列表

**接口说明**：前端需要从后端获取所有路由配置，用于展示在路由管理表格中。

**请求方式**：GET  
**请求路径**：`/api/routes`（请后端确认具体路径）

---

### 【重要】响应数据格式说明
//字段名为：id，path，component，name，status
//字段名都是string类型
//前端需要得到的是数组（由前端表格中的data属性决定），因此有数据时如下方例子所示，无数据时也需要返回一个空数组。

前端期望接收的 JSON 结构如下：

```json
有数据时：
[
  {
    "id": "user-service-route",
    "path": "/api/user/**",
    "component": "user-service",
    "name": "用户服务路由",
    “status”：“normal“
]

无数据时返回空数组
//没有数据时也要返回空数组，不能返回null或{}，否则前端无法渲染
